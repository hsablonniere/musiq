musiq

Web Components for music visualization — piano, fretboard, and more.

<mq-piano>

Renders a piano keyboard with realistic geometry. Configurable range, CSS-driven borders, stylable keys via ::part(), and per-note annotations via named slots.

<mq-piano-chord>

Displays piano chord diagrams. Pass a chord name, get a highlighted keyboard with optional note labels.

<mq-piano-scale>

Displays piano scale diagrams. Pass a scale name, get a highlighted keyboard with optional note and degree labels.

<mq-fretboard>

Full fretboard display with slot-based content placement. Scales, positions, intervals — horizontal or vertical orientation.

<mq-fretboard-chord>

Displays guitar chord diagrams. Pass a chord name, get a fretboard with finger positions and multiple voicings.

<mq-fretboard-scale>

Displays scale patterns on a fretboard. Pass a scale name, get a full-neck diagram with highlighted notes and optional labels.

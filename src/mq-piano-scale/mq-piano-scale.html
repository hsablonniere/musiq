<mq-piano-scale>

A Web Component that displays piano scale diagrams. Pass a scale name, get a highlighted keyboard.

← Back to index

Install

npm install @hsablonniere/musiq
import '@hsablonniere/musiq/mq-piano-scale';
<mq-piano-scale scale="C major"></mq-piano-scale>
